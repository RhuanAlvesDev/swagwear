#  SwagWear API

Backend de um e-commerce streetwear desenvolvido com Node.js, Express e PostgreSQL.

##  Tecnologias

* Node.js
* Express
* PostgreSQL
* Prisma ORM
* JWT
* Bcrypt

##  Funcionalidades

* Cadastro e login
* Autenticação JWT
* Controle de acesso Admin/User
* CRUD de produtos
* Busca e filtro por categoria

##  Estrutura

* Controllers
* Routes
* Middlewares
* Prisma ORM
* PostgreSQL

## ⚙️ Como executar

```bash
cd backend
npm install
npm run dev
```
